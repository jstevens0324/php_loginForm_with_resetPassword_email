A Php Login form.

Functionalities:
1. Registration Form

2. Sign In
Email ID is used to login.
A cookie for the current logged in user is set for a month.

3. Sign Out
The cookie is unset for the current logged in user.

4. Forgot Password
Forgot password sends and email with a unique 'TOKEN' url.

5. Reset Password.
From the unique 'TOKEN' url we check if the token is valid and show a form so the user can enter his new password.
